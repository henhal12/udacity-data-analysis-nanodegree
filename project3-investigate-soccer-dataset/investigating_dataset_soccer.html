Introduction¶

The dataset is a European Soccer Database from Kaggle. it contains match information of 10 different European leagues starting from 2008 to 2016. The database also carries information on team and player attributes. In this analysis, I will focus on four countries: England, Spain, Germany, and Italy.

This analysis will answer the following questions:

    1. What teams have improved the most over the time period?
    2. What team attributes do lead to the most victories?
    3. Does older player has higher overall rating than younger player?

Data Wrangling¶

Exporting Dataset Using SQL Queries¶

The datasets are exported to csv files using DB Browser for SQLite.

For the "match_full" dataset, I join the Country table, Team table, and Matches table to show all the necessary information. The query is given below:

SELECT m.id, l.name AS league_name, c.name AS country_name,
              m.season, m.stage, m.date, ht.team_long_name AS home_team,
              at.team_long_name AS away_team, m.home_team_goal AS home_goal,
              m.away_team_goal AS team_goal
JOIN League l ON m.league_id = l.id
JOIN Country c ON m.country_id = c.id
JOIN Team ht ON m.home_team_api_id = ht.team_api_id
JOIN Team at ON m.away_team_api_id = at.team_api_id
WHERE country_name IN ('England', 'Italy', 'Spain', 'Germany')
GROUP BY 1
ORDER BY 4, 5

For the "team_attribute" dataset, I join the Team table and Team Attribute table using this query:

SELECT t.team_api_id, t.team_long_name AS team_name, ta.*
FROM Team t
JOIN Team_Attributes ta ON ta.team_api_id = t.team_api_id
ORDER BY 1

For the "player_attribute" dataset, I join the Player table with Player Attribute table using this query:

SELECT *
FROM Player p
JOIN Player_Attributes pa ON p.player_api_id = pa.player_api_id

To answer the question, I create a function that change the data type for "date" table to datetime object and use it to find the player age. The recent date that I use is 2016-01-01 because the match data goes until year of 2016.

def get_age(x):
    date = x.split(" ")[0]
    born = dt.datetime.strptime(date, "%Y-%m-%d").date()
    recent_date = dt.datetime.strptime("2016-01-01", "%Y-%m-%d").date()
    return recent_date.year - born.year - ((recent_date.month, recent_date.day) < (born.month, born.day))
